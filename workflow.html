agent 团队?

你说一句大白话 → 我把它转换成一小段脚本 → 引擎按数据「代入」出许多具体指令 → 每个指令交给一个独立的 sub-agent 去干。下面拆解一次真实发生过的 81 个 agent 的项目审计。

STEP 1

你说的话

一句自然语言的需求,不含任何代码。→

STEP 2

我写的脚本

一段小 JS:一个数组 + 一个 prompt 模板 + 一个返回格式。→

STEP 3

代入生成

引擎拿数组每一项代入模板,生成 N 条具体 prompt。→

STEP 4

各自干活

N 个 agent 并行执行,各返回一份结构化结果。
